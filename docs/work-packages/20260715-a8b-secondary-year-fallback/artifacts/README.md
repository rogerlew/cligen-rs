# A8b artifacts

Prospective contract, bounded variance-reallocation coefficients, analytic and
held-out feasibility evidence, independent verification, review, and gates for
the A8b secondary year-to-year fallback decision.

- `design.md` and `feasibility-contract-v1.json` define the exact null and
  optional candidate.
- `pre-analysis-freeze-v1.json`, `pre-analysis-amendment-001.json`, and
  `pre-analysis-freeze-v2.json` preserve the prospective boundary and bounded
  fail-closed reporting amendment.
- `a8b-coefficients-v1.json` is an explicit infeasibility sentinel with no
  coefficients or use authorization.
- `a8b-analysis-v1.json`, `a8b-decision-v1.json`, and `findings.md` record the
  legacy-only terminal.
- `review.md` and `gate-results.md` record accepted closure evidence.
