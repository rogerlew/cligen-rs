# A5f0 artifacts

This directory contains the frozen design, deterministic producer/verifier,
derived attribution, narrow decision, review, and gate record for the A5f0
derived-only package.

The raw A5e0 climates remain under `target/a5e0` and are not duplicated here.
Their paths and hashes are fixed by the A5e0 matrix index. The committed A5f0
attribution is reproducible only while those retained products or exact
hash-identical recoveries are available.
