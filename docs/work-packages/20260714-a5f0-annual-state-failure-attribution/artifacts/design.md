# A5f0 Frozen Design

Date: 2026-07-14
Study class: exposed, derived-only failure attribution

A5f0 analyzes the already observed A5e0 exploratory failure. It is not a new
climate experiment and cannot repair A5e0's prospective-boundary hold. The
analysis reads only the committed A5e0 artifacts, their retained hash-indexed
matrix, and the same Daymet/base-station sources named by the coefficient
bundle. It generates no climate and fits no coefficient.

The attribution has four lenses:

1. Frozen H1 score-family decomposition identifies which evaluated behavior
   supplied positive composite degradation.
2. Daymet fit-period geometry measures how much standardized annual-feature
   covariance a single leading component can represent.
3. Candidate-state regressions test whether generated annual features respond
   in the direction and approximate scale encoded by active loadings.
4. Evaluation-period seam distances ask whether one occurrence, amount, Tmax,
   or Tmin seam is a repeatable, isolated repair target.

The lenses are descriptive and triangulating. They do not make a causal or
population claim from three exposed stations. The exact priority, thresholds,
and terminal vocabulary live in `attribution-contract-v1.json` and are frozen
with the analyzer before derived attribution is emitted.

`RETIRE-SCALAR-IID-MECHANISM` means stop investing in the exact
`a5e0_direct_annual_state_v1` direct rank-one, IID annual-state mechanism and
its loading recipe. It does not reject annual latent states, multiple factors,
persistence, regimes, or other future model structures.

`JUSTIFY-ONE-BOUNDED-ABLATION` means the retained evidence isolated exactly
one parameter seam strongly enough to warrant one future package that zeros
only that seam while keeping the mechanism, coefficients at other seams,
states, matrix, and evaluation fixed. It does not authorize that package,
implementation, model selection, or promotion.
