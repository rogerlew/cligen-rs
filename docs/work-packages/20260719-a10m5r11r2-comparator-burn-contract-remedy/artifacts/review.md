## Scientific interpretation

The A10M5R10 training-objective improvements did not transfer to realized
100-year temporal climate. Because the three variants fail similarly across
all six regimes, their common daily backbone plus residual-adapter mechanism
is the more plausible limitation than K1 versus K2 capacity or annual-loss
weighting. This is an inference from the matched portfolio, not a component-
level causal attribution.

The next test should change the temporal generative mechanism while keeping
the registered precipitation/Tmax/Tmin estimand fixed. A compact two-model
comparison—hierarchical annual/monthly latent state versus monthly-only latent
state—isolates whether explicit annual memory is necessary before reopening
solar complexity.
