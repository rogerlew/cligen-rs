# A9c3 artifact contract

The package-local design freeze is authoritative for every A9c3 outcome. The
committed A9c normalized observed objects remain in their original package;
A9c3 binds their identities rather than duplicating them. Generated daily
streams are scratch evidence and are reduced to hash-bound metrics. Fresh fit
details and compact evaluation records are retained here.

Execution order is strict:

1. `verify-scaffold.py`
2. `python3 -m research.a9c3.experiment calibrate`
3. `python3 -m research.a9c3.experiment fit`
4. `python3 -m research.a9c3.experiment evaluate`
5. `verify-a9c3.py`

`calibrate` refuses candidate artifacts. `fit` requires the calibration
identity. `evaluate` requires both and refuses any confirmation-role path.
None of these commands alters the A9c package or the production Rust runtime.
