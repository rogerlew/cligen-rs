or resource evidence is reused as a result. R13R1 preserves the complete R13
scientific surface and changes only controller/package identity plumbing.

validation, behavioral controller receipt tests, builder authority/plan
fixtures, and the staged R13 continuous-core self-test.

gates. It additionally verified that replay projects manifest entries to exact
`bytes` and `sha256` while tolerating authenticated `source_path` metadata, and
that both the R13 builder and its transitive R12R1 builder are authenticated
before import and rechecked against the authorized run commit. No authority or
