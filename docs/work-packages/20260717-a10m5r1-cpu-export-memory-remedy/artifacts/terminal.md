# A10M5R1 terminal

Terminal: `A10M5R1-EXPORT-REMEDY-READY`.

The unchanged A10 deployment contract is achievable. The canonical
Python 3.11/PyTorch export consumes roughly 0.63 GiB in a clean one-core
process; A10M5's 3.1 GiB result inherited the training parent's historical
`ru_maxrss` across fork/exec.

The remedy changes only process separation and evidence collection. It does
not change model weights, architecture, dependencies, generation output, or
the 2 GiB safeguard. A10M5 remains closed and immutable. A10M5R2 is authorized
as a new development screen retry using a clean supervisor-launched memory
worker. A10M6 remains unauthorized until that successor produces at least one
valid promotion under all frozen gates.
