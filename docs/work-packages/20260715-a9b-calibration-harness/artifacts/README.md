# A9b artifacts

This directory holds the exact predecessor/source manifests, generated
synthetic fixture evidence, requirement coverage, consolidated review, gate
results, and freeze-ready A9c handoff for A9b. No observed climate target
belongs in this package.

- `predecessor-manifest-v1.json` freezes the A9a inputs and dispatch.
- `source-manifest-v1.json` freezes 27 implementation/script records, the
  Python environment, schema libraries, and seven generated evidence files.
- `generated/` contains fixture, recovery, golden, firewall, command,
  optimizer/resource, and deterministic-replay evidence.
- `requirement-coverage.md` maps every normative MUST/MUST NOT and every A9b
  implementation unit.
- `review.md` records the consolidated six-lens review and dispositions.
- `gate-results.md` records package and repository gates.
- `a9c-handoff.md` preserves the exact next-package boundary; it does not
  authorize or scaffold A9c.
