# A8a artifacts

Prospective station selection, source acquisition, dry-regime applicability,
analytic feasibility, verification, review, and gate evidence for A8a.

- `selection-contract-v1.json`, `station-inventory-v1.json`, `panel-v1.json`,
  and `selected-parameters-v1.tar.gz` preserve the metadata-only selection.
- `analysis-contract-v1.json` fixes sources, the classifier, bootstrap,
  shortened windows, analytic mechanism, and terminal guards.
- `pre-analysis-freeze-v1.json`, `pre-analysis-amendment-001.json`, and
  `pre-analysis-freeze-v2.json` preserve the prospective boundary and the
  bounded GHCN station-list UTF-8 parser correction.
- `source-manifest-v1.json` binds the immutable archived observations.
- `a8a-analysis-v1.json`, `a8a-decision-v1.json`, and `findings.md` are the
  canonical result.
- `review.md` and `gate-results.md` record closure review and validation.
