## Architecture recommendation

The raw-feedback candidate should not continue. Seasonal temperature and low-
precipitation-quantile errors dominate the component list, while endogenous
one-step state has only modest value. A new prospective study should test an
explicit climate-normal baseline with coupled residual outputs, first without
and then with a small latent residual state. This recommendation is an
inference from the mixed residual evidence and is not an R2 selection or
authorization.
